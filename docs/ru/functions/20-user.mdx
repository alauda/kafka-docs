# Управление пользователями

Пожалуйста, настройте пользователей Kafka по мере необходимости.

<Directive type="tip" title="Предварительные условия">
  Экземпляр Kafka был создан и работает
</Directive>

## Процедура

<Tabs>
  <Tab label="Веб-консоль">
    1. Нажмите на **Kafka** в левой навигационной панели.

    2. Выберите целевое пространство имен.

    3. Нажмите на имя целевого экземпляра, чтобы перейти на страницу с деталями.

    4. На вкладке **Управление пользователями** нажмите **Создать пользователя**.

    5. Настройте параметры в соответствии со следующими инструкциями:

           <table border="">
             <tr>
               <th style={{width: '100px'}}>Параметр</th>
               <th>Описание</th>
             </tr>

             <tr>
               <td><b>Метод аутентификации</b></td>
               <td>Должен соответствовать методу аутентификации экземпляра Kafka. Если у экземпляра несколько методов аутентификации, можно выбрать один по мере необходимости.</td>
             </tr>

             <tr>
               <td><b>Разрешения</b></td>
               <td>Определяет права доступа пользователя к темам, потребителям, транзакциям и другим ресурсам.</td>
             </tr>
           </table>

    6. Нажмите **Создать**.
  </Tab>
</Tabs>

<Directive type="info" title="Примечания">
  - Пароли пользователей или сертификаты хранятся в защищенном словаре.
  - Рекомендуется регулярно обновлять пароли для пользователей, использующих аутентификацию SCRAM-SHA-512, для повышения безопасности.
</Directive>
