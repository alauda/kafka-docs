# Журнал

Понимание журналов, генерируемых в ходе работы экземпляра на уровне контейнера, таких как журналы ошибок Kafka. Правильное использование журналов может помочь вам быстро идентифицировать проблемы и устранять неисправности и исключения.

## Процедура

#

<Tabs>
  <Tab label="Web Console">
    1. Нажмите на **Kafka** в левой панели навигации.

    2. Нажмите на ***имя пространства имен***.

    3. Нажмите на ***имя экземпляра***.

    4. На вкладке **Realtime Log** установите критерии поиска.

       - Чтобы найти журналы конкретных Pods, таких как реплика Kafka: выберите Pods в разделе **Pods**, затем нажмите **Search**.

    5. (Необязательно) Нажмите <span style={{display: 'inline-block'}}>![](../../en/assets/44buttonexport.png)</span>, чтобы экспортировать журналы.
  </Tab>
</Tabs>
