# Удалить экземпляр

Удаление экземпляра не только удаляет созданные ресурсы CR, но также удаляет StatefulSet, ConfigMap, Secret, PersistentVolumeClaim и другие ресурсы, созданные оператором на основе ресурса CR.

## Процедура

<Tabs>
  <Tab label="CLI">
    - Если экземпляр не включен для сохранности данных или если вам нужно сохранить постоянные данные экземпляра, выполните следующую команду для удаления экземпляра:

    ```bash
    # удалить kafka экземпляр напрямую
    kubectl delete -n <namespace> rdskafka <name>
    ```

    - Если вы хотите удалить экземпляр вместе с его постоянным объемом, выполните следующие команды:

    ```bash
    # удалить kafka экземпляр напрямую
    kubectl delete -n <namespace> rdskafka <name>

    # удалить pvc
    kubectl delete -n <namespace> pvc -l strimzi.io/cluster=<name>,strimzi.io/kind=Kafka
    ```
  </Tab>

  <Tab label="Web Console">
    1. Нажмите на **Kafka** в левой панели навигации.

    2. Нажмите на ***имя пространства***.

    3. Выберите экземпляр для удаления из списка экземпляров и перейдите на страницу деталей экземпляра.

    4. В верхнем правом углу страницы **Детали экземпляра** нажмите кнопку **Действия**, чтобы развернуть меню функций.

    5. Нажмите на **Удалить** в меню функций, и появится диалоговое окно подтверждения.

    6. Для экземпляров с включенной сохранностью данных вы можете отметить "Удалить Persistent Volume Claim", чтобы освободить место, или оставить его неотмеченным, чтобы сохранить данные.

    7. В всплывающем окне введите имя экземпляра для удаления.

    8. Нажмите **Удалить**.
  </Tab>
</Tabs>
