# Управление темами

Тема — это тематический набор данных. Производители записывают информацию в тему, а потребители читают информацию из темы. Темы обеспечивают распределенное хранение сообщений через разделы.

## Создание темы

<Tabs>
  <Tab label="Веб-консоль">
    1. Нажмите на **Kafka** в левом навигационном меню.
    2. Нажмите на имя целевого пространства имен.
    3. Нажмите на имя целевого экземпляра.
    4. На вкладке **Управление темами** нажмите на **Создать тему**.
    5. Заполните информацию о теме.
       **Примечание:** Если поле **Имя темы** не заполнено, имя темы будет соответствовать имени создаваемого ресурса Kubernetes (поле **Имя**).
    6. Нажмите на **Создать**.
  </Tab>
</Tabs>

> Пример реплики раздела: реплика раздела 0 распределена между Брокером 2 (Ведущим) и Брокером 1 (Подчиненным)

## Синхронизация темы

<Tabs>
  <Tab label="Веб-консоль">
    1. Нажмите на **Kafka** в левом навигационном меню.
    2. Нажмите на имя целевого пространства имен.
    3. Нажмите на имя целевого экземпляра.
    4. На вкладке **Управление темами** нажмите на **Синхронизировать тему**.
    5. В всплывающем окне выберите тему со статусом **Работает**.
    6. Нажмите на **Синхронизировать**.
  </Tab>
</Tabs>

## Расширение разделов темы

<Tabs>
  <Tab label="Веб-консоль">
    1. Нажмите на **Kafka** в левом навигационном меню.
    2. Нажмите на имя целевого пространства имен.
    3. Нажмите на имя целевого экземпляра.
    4. На вкладке **Управление темами** нажмите на имя целевой темы.
    5. Нажмите на **Действия > Расширить разделы**.
    6. Установите количество новых разделов и выберите Брокеров.
    7. Нажмите на **Расширить**.
  </Tab>
</Tabs>

## Управление разделами темы

<Tabs>
  <Tab label="Веб-консоль">
    1. Нажмите на **Kafka** в левом навигационном меню.
    2. Нажмите на имя целевого пространства имен.
    3. Нажмите на имя целевого экземпляра.
    4. На вкладке **Управление темами** нажмите на имя целевой темы.
    5. Нажмите на **Действия > Управление разделами**.
    6. Настройте расположение Брокеров для реплик каждого раздела.
    7. Нажмите на **Применить**.
  </Tab>
</Tabs>
