# Запуск и остановка экземпляра

Функция одноразового запуска и остановки удовлетворяет повседневным потребностям в разработке и обслуживании, позволяя вам быстро приостанавливать или восстанавливать экземпляры промежуточного программного обеспечения. В таких сценариях, как запланированное время простоя, высокая нагрузка на узлы и среды разработки тестирования, вам может потребоваться временно освободить ресурсы ЦП и памяти, не удаляя экземпляры. Функция одноразового запуска и остановки помогает вам экономить ресурсы и гибко управлять экземплярами промежуточного программного обеспечения, тем самым снижая затраты и повышая эффективность.

## Остановка экземпляра

### Предварительные требования

1. Версия экземпляра - `2.5.0`, `2.7.0` или `2.8.2`.
2. Экземпляр находится в статусе **Обработка**, **Запуск**, **Аномалия** или **Неизвестно**.

### Процедура

<Tabs>
  <Tab label="Веб-консоль">
    **Совет**: рекомендуется выполнить полное резервное копирование перед остановкой экземпляра, чтобы предотвратить потерю данных.

    1. Нажмите **Kafka** в левой навигационной панели.

    2. Нажмите ***Имя пространства***.

    3. Нажмите ***Имя экземпляра***.

    4. Нажмите **Действия** > **Остановить**, и подтвердите.
  </Tab>
</Tabs>

### Функции, недоступные после остановки

| Функция                    | Описание                                                    |
| --------------------------- | ----------------------------------------------------------- |
| **Терминальная консоль**    | Терминальная консоль не может быть использована.            |
| **Действия**                | Невозможно перезапустить, обновить или обновить конфигурацию расписания. |
| **Топология**               | Невозможно просмотреть страницу топологии.                 |
| **Конфигурация параметров** | Невозможно обновить параметры и экспортировать параметры.   |
| **Управление пользователями**| Невозможно создать пользователей и просмотреть детали пользователей. |
| **Управление темами**       | Невозможно создать темы и просмотреть детали темы.         |
| **Группы потребителей**     | Невозможно просмотреть страницу группы потребителей.       |
| **Инспекция**               | Результат инспекции для текущего экземпляра недействителен. |
| **Метод доступа**           | Информация о маршрутизации может быть недоступна.         |

## Запуск экземпляра

### Предварительные требования

Экземпляр находится в статусе **Остановка** или **Остановлен**.

### Процедура

<Tabs>
  <Tab label="Веб-консоль">
    1. Нажмите **Kafka** в левой навигационной панели.

    2. Нажмите ***Имя пространства***.

    3. Нажмите ***Имя экземпляра***.

    4. Нажмите **Действия** > **Запустить**, и подтвердите.
  </Tab>
</Tabs>
