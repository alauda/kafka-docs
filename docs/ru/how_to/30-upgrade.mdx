# Обновление патч-версии

В связи с необходимостью исправления уязвимостей или выпуска новых функций, могут производиться обновления патч-версии для программного обеспечения промежуточного слоя. В процессе обновления патч-версии Pods будут перезапущены по одному, что может вызвать временное прерывание в обслуживании. Поэтому рекомендуется выбирать подходящее временное окно для обновления. Когда доступна патч-версия для обновления, рекомендуется выполнить обновление как можно скорее, чтобы применить последние исправления для компонентов.

## Установка стратегии обновления патч-версии

### Процедура

<Tabs>
  <Tab label="Веб-консоль">
    1. Перейдите на страницу сведений о экземпляре.

    2. Нажмите **Стратегия обновления патч-версии**.

    3. Выберите **Ручное** или **Автоматическое**.

    4. Нажмите **Обновить**, чтобы завершить настройки.
  </Tab>
</Tabs>

## Обновление патч-версии

### Предварительные условия

Когда патч-версия доступна, на странице списка экземпляров будет отображаться значок обновления, и на странице сведений о экземпляре появится сообщение о необходимости обновления.

### Процедура

<Tabs>
  <Tab label="Веб-консоль">
    1. Перейдите на страницу сведений о экземпляре.

    2. Нажмите **Обновить сейчас** рядом с сообщением о необходимости обновления.

    3. Прочитайте сообщение об обновлении, чтобы понять изменения версии для компонентов и плагинов.

    4. Нажмите **Обновить сейчас**.

    5. Дождитесь изменения статуса экземпляра на "Работающий", что указывает на успешное обновление.
  </Tab>
</Tabs>

## Пакетное обновление патч-версии

### Предварительные условия

Пользователь должен иметь права администратора платформы.

### Процедура

<Tabs>
  <Tab label="Веб-консоль">
    1. В левом навигационном меню нажмите **Управление обновлениями**.

    2. Выберите необходимого **оператора**, перейдите на страницу сведений об операторе.

    3. Выберите экземпляры, которые необходимо обновить, и нажмите **Пакетное обновление**.

    4. Прочитайте сообщение об обновлении, чтобы понять изменения версии для компонентов и плагинов.

    5. Нажмите **Обновить сейчас**.

    6. Дождитесь изменения статуса экземпляра на "Работающий", что указывает на успешное обновление.
  </Tab>
</Tabs>
