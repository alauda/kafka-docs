# Обновление

Сервис стриминга Alauda для Kafka будет выполнять обновления на основе заданной стратегии обновления:

- **Автоматическое** : Автоматические обновления активируются немедленно после обнаружения новых версий компонентов.
- **Ручное** : Требуется ручное одобрение перед началом процесса обновления.
