# 日志

从容器层面了解实例运行过程中的日志，例如 Kafka 错误日志。妥善使用日志能帮助您快速定位问题，处理故障和异常。

## 操作步骤
#
<Tabs>

<Tab label="Web Console">

1. 在左侧导航栏中，单击 **Kafka**。

2. 单击 ***命名空间的名称***。

3. 单击 ***实例名称*** 。
    
4. 在 **实时日志** 页签中，设置查找条件。

    - 查找特定 Pods，即 Kafka 副本的日志：在 **Pods** 处，选择一个 Pods，并单击 **搜索**。

5. （可选）单击 <span style={{display: 'inline-block'}}>![](../assets/44buttonexport.png)</span>，导出日志

</Tab>
</Tabs>
