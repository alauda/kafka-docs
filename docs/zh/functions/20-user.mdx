# 用户管理

请按需配置 Kafka 用户。

<Directive type="tip" title="前提条件">
已创建 Kafka 实例并处于运行状态
</Directive>

## 操作步骤

<Tabs>
<Tab label="Web Console">

1. 在左侧导航栏中，单击 **Kafka**。
2. 选择目标命名空间。
3. 单击目标实例名称进入详情页。
4. 在 **用户管理** 页签中，单击 **创建用户**。
5. 参考以下说明配置参数：

    <table border="">
        <tr>
            <th style={{width: '100px'}}>参数</th>
            <th>说明</th>
        </tr>
        <tr>
            <td><b>认证方式</b></td>
            <td>需与 Kafka 实例的认证方式保持一致。实例存在多种认证方式时，可按需选择其中一种。</td>
        </tr>
        <tr>
            <td><b>权限</b></td>
            <td>定义用户对主题、消费者、事务等资源的访问权限。</td>
        </tr>
    </table>

6. 单击 **创建**。

</Tab>
</Tabs>

<Directive type="info" title="注意事项">
- 用户密码或证书存储在保密字典中
- 当使用SCRAM-SHA-512认证的用户时，建议定期更新密码以增强安全性
</Directive>
