# 删除实例

删除实例不仅会移除所创建的 CR 资源，还会删除 Operator 根据 CR 资源创建的 StatefulSet、ConfigMap、Secret、PersistentVolumeClaim 以及其他资源。

## 步骤

<Tabs>

<Tab label="CLI">

* 如果实例没有开启持久化，或者您需要保留实例的持久化数据，请执行以下命令删除实例：

```bash
# delete kafka instance directly
kubectl delete -n <namespace> rdskafka <name>
```

* 如果您希望删除实例时同时删除其持久卷申请，请执行以下命令：

```bash
# delete kafka instance directly
kubectl delete -n <namespace> rdskafka <name>

# patch delete pvc
kubectl delete -n <namespace> pvc -l strimzi.io/cluster=<name>,strimzi.io/kind=Kafka
```

</Tab>

<Tab label="Web Console">

1. 在左侧导航栏中，单击 **Kafka**。

2. 单击 ***命名空间名称***。

3. 在实例列表中选择要删除的实例，并进入实例详情页面。

4. 在 **实例详情** 页面的右上角，点击 **操作** 按钮，展开功能菜单。

5. 点击功能菜单中的 **删除**，会弹出确认对话框。

6. 对于开启了数据持久化的实例，您可以勾选“删除持久卷申明”以释放存储，或不勾选以保留数据。

7. 在弹出窗口中，输入要删除的实例名称。

8. 点击 **删除**。

</Tab>

</Tabs>
